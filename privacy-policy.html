Privacy Policy

Smart Form Autofill · Last updated: March 2, 2026

Smart Form Autofill ("the extension") is a Chrome extension that remembers and auto-fills form field values when you revisit websites.

Data Collection

The extension stores:

  • Form field values you enter (e.g., name, email, address) when you leave a field or change a dropdown
  • Per-field override preferences (whether to always store or never store specific fields)

Data Storage & Transmission

  • All data is stored locally using Chrome's storage API (chrome.storage.sync)
  • Data syncs across your Chrome browsers where you are signed in, via Chrome Sync
  • No data is transmitted to our servers or any third party
  • We do not collect, receive, or have access to your stored form data

Sensitive Data

The extension excludes sensitive fields by default, including: passwords, credit cards, CVV, SSN, PIN, OTP, and similar. You can override this per field via right-click.

Data Deletion

You can clear all stored data at any time via the extension popup ("Clear all stored data").

Contact

For questions about this privacy policy, open an issue on GitHub.
